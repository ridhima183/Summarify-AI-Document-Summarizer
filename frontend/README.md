## ☕ About

**Summarify** is an AI-powered web application that transforms lengthy documents into concise, coherent summaries within seconds.

Whether it's lecture notes, research papers, reports, or articles — Summarify helps you grasp the essential meaning without turning every page. Built on a transformer-based **BART** model for abstractive summarization, with **LongDoc** techniques for handling extended text and **ROUGE** scoring to keep quality in check.

Designed with simplicity, speed, and a cozy reading-nook feeling in mind.

---

## ✨ Features

- 🍂 Upload PDF, DOCX, and TXT files
- 📖 AI-generated abstractive summaries (powered by BART)
- 📝 Bullet-point summaries
- 🎯 Key information extraction
- 📚 Important sentence detection
- 📏 ROUGE-based summary evaluation
- ⚡ Fast document processing, even for long documents
- 🎨 Clean and responsive interface
- 🌙 A beautiful, cozy reading experience

## 🌿 How It Works

```text

        │
        ▼
   📖  Extract Text        (PyMuPDF / python-docx)
        │
        ▼
   🧹  Clean & Preprocess
        │
        ▼
   🤖  Summarize with BART  (+ LongDoc for long inputs)
        │
        ▼
   📏  Evaluate with ROUGE
        │
        ▼
   📚  Display Results
```

## 🍂 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/Summarify.git
cd Summarify
```

### 📦 Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 🎨 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The app should now be running locally — happy summarizing! ☕
